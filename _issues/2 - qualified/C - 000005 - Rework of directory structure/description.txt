New directory and project structure of eFaps:

|- _issue
|    |- 0 - open numbers
|    |- 1 - new
|    |- 2 - qualified
|    |- ........
|- addon
|    |- webapp
|    |- directory (WebDAV, FTP, ...)
|    |    |- maven-plugin (um in DB zu installieren)
|    |    |- webapp (um Servlet, etc. zu definieren)
|    |- lucene
|         |- maven-plugin (um in DB zu installieren)
|         |- webapp (um Servlet, etc. zu definieren)
|- example
|    |- application (um alle applikation zu installieren, mit jetty:run zu starten, ...)
|    |- derby (start / stop db)
|    |- shell (um eFaps Ÿber eine shell zu starten, mit uberjar)
|- framework
|    |- kernel
|    |- kernel-install
|    |- maven
|- module
     |- teamwork (um in DB zu installieren)
     |- addresswork (um in DB zu installieren)
